# Sitelen Pona

This plugin allows you to toggle between regular and Sitelen Pona font for notes written in Toki Pona.

## Usage

Once enabled, you can use the built-in `Sitelen Pona: Toggle` command via Command Palette. Or you can manually add `sitelen-pona` to the `cssclasses` property of your file.

The line you're currently editing will be shown in regular font, to make editing easier.

You can also prefix a line with `!!` to force it back to the regular font.
